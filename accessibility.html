Accessibility Statement

Last Updated: July 2026

Agent Peel (agentpeel.com) is committed to digital accessibility. We continuously work to ensure that our website is usable and accessible to all visitors, including individuals with disabilities, in accordance with the Equal Rights for Persons with Disabilities Regulations (Accessibility Adjustments for Service), 5773-2013, and the Israeli Standard SI 5568 (which is based on the Web Content Accessibility Guidelines WCAG 2.1 Level AA).

Key Accessibility Features

  • Keyboard Navigation: Full support for standard navigation using the Tab, Shift+Tab, and Enter keys.
  • Alternative Text: Comprehensive alternative text (Alt Text) provided for all meaningful images, icons, and diagrams.
  • Color Contrast: Text colors designed to meet or exceed standard contrast ratios (at least 4.5:1 for regular text) against background colors.
  • Semantic Markup: Proper hierarchical heading structure (H1–H3) for predictable, logical navigation using screen readers and assistive technologies.
  • Browser Zoom: Fully compatible with built-in browser zoom tools to increase text size without breaking layout structures.
  • ARIA Descriptions: Custom ARIA attributes added to dynamic components like cookie banners, contact forms, and FAQ accordions.

Known Limitations

Despite our best efforts, certain parts of this website might not yet be fully optimized, or are hosted by third-party services that we cannot control directly (such as Paddle’s checkout interface). We are continuously monitoring the site to resolve any newly discovered accessibility barriers.
