How Agent Peel Works

An innovative learning loop that builds real critical thinking and scientific inquiry.

📖

1. Read the Clues

The story halts at a critical juncture. To find the suspect, a hidden code or substance must be scientifically analyzed.

🧪

2. Conduct the Experiment

Using standard kitchen items (vinegar, baking soda, red cabbage), kids run a safe experiment to generate real-world data.

🔓

3. Solve the Mystery

The outcome of the experiment reveals the scientific culprit, allowing Agent Peel to close the case file successfully!
